
# Better-KeyBoard

减轻小手指负担，提升键入效率，同时降低错误键入机率。

键位变化很少，方便快速适应。

[demo](https://googee.github.io/Better-KeyBoard/dist)


## 变动：

- 将 4 个常用键移动到键盘中央
- 将 Ctrl 键交由大拇指负责
- 增加 MO(1) 键
